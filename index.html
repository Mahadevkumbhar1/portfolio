2. TO-DO LIST

I started my adventure by creating the HTML structure of the "To-Do List." I began with a basic layout, consisting of a header, an input field to add tasks, a button to submit tasks, and an empty unordered list to display the tasks. Next, I delved into the world of CSS, styling the elements to create an attractive and user-friendly interface. I chose a calming color scheme and used elegant fonts to enhance the visual appeal. I added subtle animations to the buttons to provide a delightful user experience. Once the design was in place, I moved on to the heart of the "To-Do List" - JavaScript. I began by selecting the necessary elements from the HTML using JavaScript DOM manipulation. This allowed me to dynamically interact with the elements and update the content without having to refresh the page.
